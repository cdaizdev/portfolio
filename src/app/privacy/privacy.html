Política de Privacidad

En Mi portfolio, valoramos tu privacidad. Esta política explica cómo recopilamos, usamos y protegemos tu información personal.

1. Información que recopilamos

No realizamos recopilación de ningún tipo de datos. La web se aloja en un servidor de Github el cual no recoge información

3. Seguridad

Implementamos medidas de seguridad para proteger tu información contra accesos no autorizados. Esta web no contiene formularios ni le debe permitir enviar información de ninguna clase. La web se revisa constantemente y no debería ver contenido que redirija a sitios externos o de poca confianza, salvo que se produzcan vulnerabilidades del servidor de Github de día 0.
